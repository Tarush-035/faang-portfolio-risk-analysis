# FAANG Portfolio Risk Analysis

Author: Tarooshna Singh Chauhan  
Institution: BITS Pilani

📄 View the report: [FAANG Portfolio Risk Report](FAANG_Portfolio_Risk_Report.pdf)

## Overview

This project analyzes the risk characteristics of FAANG stocks 
(Facebook, Amazon, Apple, Netflix, Google) using historical 
monthly returns.

The analysis evaluates:

• Volatility  
• Systematic risk (Beta relative to the S&P 500)  
• Risk-adjusted performance using the Treynor Ratio  
• Downside risk using Value at Risk (VaR)

## Methodology

- Monthly stock returns calculated from historical prices
- Beta estimated relative to the S&P 500 market index
- Treynor ratio used to evaluate risk-adjusted returns
- Historical Value at Risk calculated at 95% and 99% confidence levels

## Tools

- Microsoft Excel
- Yahoo Finance historical data
